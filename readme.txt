[ 스프링 웹 프로젝트]
- 소개 : 스프링의 기능을 배우고 다양한 개념을 학습 및 실습

- 목표 : 스프링 프레임워크를 활용한 웹 애플리케이션 개발의 핵심 원칙과 사례를 체계적으로 학습하기 위한 교육용 리포지토리
           실무 중심의 스프링 기술 습득(Spring MVC, Spring Security,..., RESTful API, 테스트 주도 개발)
           교육 커리큘럼과 실습의 통합(의존성 주입, AOP, 트랜잭션 관리 등)
